### 3.1 共鸣数据一键导出

共鸣数据迁移功能为共鸣导出功能的扩展。用户可选择将共鸣中数据进行导出到本地，功能入口在"我的"、注销流程中。

当前版本的一键导出功能，一键全选导出功能：

- 导出账号共鸣"全部"中的所有音频及字幕、台本、封面文件
- 导出文件为一个压缩包
- 压缩包中，一个音频条目数据为一个目录，目录名为音频名去掉后缀，目录下包含音频及对应的字幕、台本、封面文件

### 4. 注册登录

服务端内容可参见v1回账号注册登录注销需求0909。

#### 4.1 注册账号

- 当前注册仅支持手机号注册
  - 一个手机号仅支持注册一个账号
  - 仅支持中国大陆手机号注册
- 用户在发送验证码请求成功之后，冷却时间内不允许用户频繁操作（APP限制发送验证码按键无法点击，服务端限制验证码发送）
- 注册需要用户同意用户协议、隐私政策
- 注册异常
  - 账号已注册
  - 手机号异常
  - 账号已被锁定，请x分钟后重试
  - 请求过于频繁，请稍后重试
  - 验证码错误，错误5次以上将会锁定账号，当前错误x次
  - 两次密码不一致

#### 4.2 登录

- 登录方式
  - 密码登录
  - 验证码登录
- 其他入口
  - 忘记密码
  - 注册账号
  - 用户协议
  - 隐私政策
- 登录需要用户同意用户协议、隐私政策
- 已注册验证过的账号，需要设置密码才能登录
  - 已注册未设置用户密码的账号，只能通过验证码登录，且验证码验证后跳转密码设置，完成密码设置才能进入主页（可见v1中账号为初始化状态，需要设置密码才能完成登录回账号注册登录注销需求0909）
- 登录异常
  - 账号未注册
  - 验证码错误，错误5次以上将会锁定账号，当前错误x次
  - 密码错误，错误5次以上将会锁定账号，当前错误x次
  - 账号已被锁定，请x分钟后重试
  - 请求过于频繁，请稍后重试

### 5. 用户权限

控制器和剧情页面需要对账号做校验才能访问。控制器页面需要验证账号实名认证是否年满十八周岁；剧情模块需要验证账号实名认证是否年满十八周岁，以及账号是否拥有对应的权限。

#### 5.1 实名认证

用户控制页面、剧情体验访问（从主页入口进入）的条件是需要完成实名认证年满十八周岁（中华人民共和国身份证）。

- 对于未实名认证的账号，用户点击进入控制页面、剧情体验时，先进入实名认证流程
- 对于已实名认证的账号，若用户已经认证完成年满十八周岁，则取决于账号所拥有的页面权限
- 对于已实名认证的账号，若用户未年满十八周岁，则弹窗提示用户是否刷新，用户选择刷新则重新验证是否年满十八周岁，否则留在当前页面（主页）
  - 若重新验证已年满十八周岁，则进入页面
  - 若仍未年满十八周岁，弹窗拒绝用户使用对应功能
- 年满十八周岁的计算方式为：当前日期为用户身份证号出生日期的十八年后同一日及以后

#### 5.2 页面权限

剧情模块需要账号拥有页面权限才能访问。

- 访问剧情模块需要PAGE_LIVE2D权限
- 访问剧情内页需要对应的PAGE_STORY_ORVEN_1_1到PAGE_STORY_VARYN_1_6权限
- 权限码

| permission_id | permission_code | description |
|---|---|---|
| 1100 | PAGE_LIVE2D | 访问剧情模块 |
| 1101 | PAGE_STORY_ORVEN_1_1 | 访问剧情1-1 |
| 1102 | PAGE_STORY_ORVEN_1_2 | 访问剧情1-2 |
| 1103 | PAGE_STORY_SOLAN_1_3 | 访问剧情1-3 |
| 1104 | PAGE_STORY_SOLAN_1_4 | 访问剧情1-4 |
| 1105 | PAGE_STORY_VARYN_1_5 | 访问剧情1-5 |
| 1106 | PAGE_STORY_VARYN_1_6 | 访问剧情1-6 |

#### 5.3 权限解锁

- 用户连接过一次OMAO设备后，即可获得5.2中剧情模块所有权限
- 剧情解锁逻辑可见v2-剧情模块
- 获取权限不影响后台回收权限/失效账号已获取权限
