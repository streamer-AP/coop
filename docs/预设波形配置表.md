# 预设波形配置表

## 数据库表结构

### `Waveforms` 波形主表

| 字段 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| id | int | autoIncrement | 主键 |
| name | text | - | 波形名称 |
| channel | text | - | 通道：`swing`（摆动）/ `vibration`（振动） |
| durationMs | int | 8000 | 波形时长（毫秒） |
| signalIntervalMs | int | 200 | 蓝牙信号发送间隔（毫秒） |
| signalDelayMs | int | 0 | 信号延迟（毫秒） |
| isBuiltIn | bool | true | 是否为内置预设 |
| createdAt | datetime | now | 创建时间 |
| updatedAt | datetime | now | 更新时间 |

### `WaveformKeyframes` 关键帧表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | int | 主键 |
| waveformId | int | 外键 → Waveforms.id |
| timeMs | int | 时间点（毫秒） |
| value | int | 强度值 |

- 唯一约束：(waveformId, timeMs)

### `FavoriteSlots` 常用槽位表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | int | 主键 |
| channel | text | swing / vibration |
| page | int | 页码 0-2 |
| slotIndex | int | 槽位 0-3 |
| waveformId | int | 外键 → Waveforms.id |

- 唯一约束：(channel, page, slotIndex)
- 容量：每通道 3 页 × 4 槽 = 12 个收藏位

### `UsageLogs` 使用日志表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | int | 主键 |
| startTime | datetime | 开始时间 |
| signalMode | text | 信号模式 |
| waveformId | int | 波形 ID |
| intensityLevel | int | 强度等级 |
| durationMs | int | 使用时长（毫秒） |
| deviceModel | text? | 设备型号（可选） |
| deviceSerial | text? | 设备序列号（可选） |
| isSynced | bool | 是否已同步（默认 false） |

## Domain 模型

### Waveform

```dart
enum WaveformChannel { swing, vibration }

class Waveform {
  int id;
  String name;
  WaveformChannel channel;
  int durationMs;         // 默认 8000
  int signalIntervalMs;   // 默认 200
  int signalDelayMs;      // 默认 0
  bool isBuiltIn;         // 默认 true
  List<WaveformKeyframe> keyframes;
}
```

### WaveformKeyframe

```dart
class WaveformKeyframe {
  int timeMs;
  int value;
}
```

### FavoriteSlot

```dart
class FavoriteSlot {
  WaveformChannel channel;
  int page;      // 0-2
  int index;     // 0-3
  int waveformId;
}
```

## 备注

- 每个波形为**单通道**（swing 或 vibration），v6 迁移时从双通道拆分
- 蓝牙信号优先级：story（剧情）> resonance（同步共鸣）> preset（预设波形）
- 当前无内置种子数据，数据库创建后波形表为空
