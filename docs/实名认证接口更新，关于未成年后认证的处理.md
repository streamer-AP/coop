【实名认证接口】

（目前没有校验同一个身份证注册多个账号的情况）



# 认证接口

POST /userAccounts/realNameVerify
BODY参数（正常情况）

~~~json
{
    "residentIdCard":"371521198411051559",
    "residentIdCardName": "陈庆山"
}
~~~



### 身份证号乱输,正则如下

~~~json
{
    "residentIdCard":"1",
    "residentIdCardName": "陈庆山"
}
~~~



^[1-9]\d{5}(18|19|20)\d{2}(0[1-9]|1[0-2])(0[1-9]|[12]\d|3[01])\d{3}[\dXx]$

~~~json
{
    "code": 1006,
    "message": "身份证号格式不正确",
    "data": null,
    "timestamp": 1768022349275
}
~~~



### 姓名不合法

~~~json
{
    "residentIdCard":"371521198411051559",
    "residentIdCardName": "陈"
}
~~~



~~~json
{
    "code": 1003,
    "message": "姓名格式不正确",
    "data": null,
    "timestamp": 1772898684917
}
~~~



### 身份证不对

~~~json
{
    "residentIdCard":"371521198411051555",
    "residentIdCardName": "陈庆山"
}
~~~



~~~json
{
    "code": 1002,
    "message": "身份证信息格式不正确",
    "data": null,
    "timestamp": 1772897763089
}
~~~



### 身份证对，姓名不匹配

~~~json
{
    "residentIdCard":"371521198411051559",
    "residentIdCardName": "张三"
}
~~~



~~~json
{
    "code": 1004,
    "message": "身份信息不一致",
    "data": null,
    "timestamp": 1772899195106
}
~~~



### 身份证对，姓名匹配，成年人

~~~json
{
    "code": 200,
    "message": "实名认证成功",
    "data": {
        "isIdentityMatch": true,
        "isAdult": true,
        "verifyStatus": "success",
        "message": "验证通过",
        "residentStatus": "1"
    },
    "timestamp": 1768020681787
}
~~~



### 身份证对，姓名匹配，未成年人（实名信息已记录），发起再次提交，提交不通过（还未成年）

~~~json
{
"auditCheck":"1"
}
~~~

返回

~~~json
{
    "code": 1007,
    "message": "您仍然未成年",
    "data": null,
    "timestamp": 1772899273648
}
~~~

### 身份证对，姓名匹配，未成年人（实名信息已记录），发起再次提交，提交通过（成年）

~~~json
{
"auditCheck":"1"
}
~~~

返回

~~~json
{
    "code": 1004,
    "message": "实名验证更新成功",
    "data": null,
    "timestamp": 1772899328194
}
~~~





### 当前实名已验证，成年，重新验证提示如下（无论当前信息正确与否）

~~~json
{
    "residentIdCard": "371521198411051559",
    "residentIdCardName": "张三"
}
~~~

~~~json
{
    "code": 1000,
    "message": "您实名验证已成功",
    "data": null,
    "timestamp": 1772899926662
}
~~~





## 当前实名已验证，成年，重新验证，未成功不覆盖

~~~json
{
    "residentIdCard": "371521198411051559",
    "residentIdCardName": "张三",
    "modify": "1"
}
~~~

~~~json
{
    "code": 1004,
    "message": "身份信息不一致",
    "data": null,
    "timestamp": 1772899960840
}
~~~

### 当前实名已验证，成年，重新验证，身份证成功覆盖（认证成功且未成年也会覆盖）

~~~json
{
    "code": 200,
    "message": "实名认证成功",
    "data": {
        "isIdentityMatch": true,
        "isAdult": true,
        "verifyStatus": "success",
        "message": "验证通过",
        "residentStatus": "1"
    },
    "timestamp": 1772900032314
}
~~~







# 查询

GET /auth/getCurrentUserInfo

返回

~~~json
{
    "code": 200,
    "message": "Success",
    "data": {
        "id": 20,
        "userId": 2,
        "omaoId": "OM1763224205518010",
        "mobile": "13012341234",
        "status": 1,
        "accountType": 1,
        "userType": 2,
        "userName": "用户1234",
        "residentStatus": "1"
    },
    "timestamp": 1772900091350
}
~~~





residentStatus 0 未实名认证 

residentStatus 1 已实名认证且成年

residentStatus 2 已实名认证且未成年
